
#Lunar Slingshot
===============

##Controls:

###Red:
	-Turn left:   s
	-Turn right:  f
	-Thrust:      d
	-Fire:        e

###Blue:
	-Turn left:   j
	-Turn right:  l
	-Thrust:      k
	-Fire:        i

###Exit:          
	-Esc
